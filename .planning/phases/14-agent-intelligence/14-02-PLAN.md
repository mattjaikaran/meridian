<objective>
Build context window awareness for optimal prompt sizing.

Different models have different context windows:
- Claude Opus 4.6: 1M tokens
- Claude Sonnet: 200k tokens
- Subagents: typically 200k

Adapt prompt generation:
1. Detect available context from config or model metadata
2. Allocate budget: 10% system, 20% plan, 50% code context, 20% tools/reserve
3. For large context: include full file contents, more history, all decisions
4. For small context: use file summaries, recent decisions only, key excerpts
5. Emit warning if generated prompt exceeds 80% budget

Budget allocation:
```
total = detect_context_size()
system_budget = total * 0.10
plan_budget = total * 0.20
code_budget = total * 0.50
reserve = total * 0.20
```
</objective>
