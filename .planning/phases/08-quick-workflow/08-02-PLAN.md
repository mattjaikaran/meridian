<objective>
Build /meridian:do as a freeform text router that maps natural language to the right /meridian:* command.

User says "check what's next" → routes to /meridian:next
User says "add a note about auth" → routes to /meridian:note
User says "show me the dashboard" → routes to /meridian:dashboard

Implementation:
1. Build a command registry from skills/*/SKILL.md metadata (name, description, keywords)
2. Score input against registry using keyword overlap + fuzzy description matching
3. If single high-confidence match: show match and confirm
4. If ambiguous: show top 3 candidates and ask user to pick
5. If no match: suggest /meridian:help
</objective>
