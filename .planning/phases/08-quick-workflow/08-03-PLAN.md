<objective>
Build /meridian:note for zero-friction idea capture during work sessions.

Subcommands:
- `append <text>` — Add a timestamped note to .meridian/notes.md
- `list` — Show all notes with IDs
- `promote <id>` — Convert note to a quick_task in the DB

Notes are stored as simple markdown in `.meridian/notes.md`:
```
## Notes

- [N001] 2026-03-20 10:30 — Consider adding batch mode to executor
- [N002] 2026-03-20 11:15 — The review skill should support --quick flag
```

Promote copies the note text into a quick_task record and marks the note as promoted.
</objective>
