    3. Add safe_update() function:
       ```python
       def safe_update(conn, table: str, row_id, updates: dict, id_column: str = "id") -> None:
           allowed = ALLOWED_COLUMNS.get(table)
           if allowed is None:
               raise ValueError(f"Unknown table: {table}")
           invalid = set(updates.keys()) - allowed
           if invalid:
               raise ValueError(f"Invalid columns for {table}: {invalid}")
           if not updates:
               return
           set_clause = ", ".join(f"{k} = ?" for k in updates)
           values = list(updates.values()) + [row_id]
           conn.execute(f"UPDATE {table} SET {set_clause} WHERE {id_column} = ?", values)
       ```

       - Note: The per-function `allowed` sets in update_project/update_phase/update_plan filter **kwargs** from the caller. safe_update's ALLOWED_COLUMNS is the **security** layer validating against schema. Both layers serve different purposes.
