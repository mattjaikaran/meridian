<objective>
Add stub detection to the quality gates module.

After a plan's subagent finishes, scan all modified files for placeholder code patterns:
- `TODO`, `FIXME`, `XXX`, `HACK` comments
- `raise NotImplementedError`
- Functions with only `pass` as body
- `# placeholder`, `# stub`, `# temporary`
- `...` (Ellipsis) as function body

Implementation:
1. Accept a list of file paths (from plan's files_modified or git diff)
2. Read each file and scan line-by-line against patterns
3. Return list of {file, line, pattern, context} dicts
4. Integrate into verification step as a warning
</objective>
