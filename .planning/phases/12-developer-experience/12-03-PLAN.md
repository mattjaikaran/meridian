<objective>
Build a discussion audit trail that captures the reasoning behind decisions.

When decisions are made during planning, log the full context:
```markdown
## [DISC-001] 2026-03-20 — Authentication approach

**Topic:** How to handle user authentication
**Options considered:**
1. JWT with refresh tokens — stateless, scalable
2. Session cookies — simpler, more secure by default
3. OAuth2 only — delegate to identity provider

**Decision:** Option 1 (JWT with refresh tokens)
**Rationale:** Need stateless auth for API-first architecture
**Decision ID:** DEC-005
```

Storage: `.meridian/DISCUSSION-LOG.md` (append-only markdown)
Linking: Each discussion entry references the decision_id it produced
</objective>
